# Dashboard Operacional de Despacho

Estudio de tiempos y movimientos para análisis de causa raíz de retrasos en despacho.

## Archivos
- `dashboard_despacho.py` — script principal que genera el dashboard
- `Plantilla_Toma_de_Tiempos_-_Bitácora_Macro.csv` — datos de llegadas de vehículos
- `Plantilla_Toma_de_Tiempos_-_Detalle.csv` — detalle minuto a minuto del proceso

## Cómo usar
1. Instala las dependencias: `pip install pandas plotly`
2. Actualiza los CSV con los nuevos datos
3. Ejecuta: `python dashboard_despacho.py`
4. Se abre el dashboard automáticamente en el navegador
